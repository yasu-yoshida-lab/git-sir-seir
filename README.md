
# ファイル構造に関して

#### README.md  
このファイルです。各ファイルやフォルダ構造の説明が書いてあります。

#### makefile
C++ および Python を実行するためのスクリプト実行の際には適宜、コンパイラを選んでください。

* C++ でシミュレーションを行う場合 `make cpp`

* Python でシミュレーションを行う場合 `make python`

ただし、結果の図（png）の出力に関しては C++ および Python のシミュレーションに関するコードで Matplotlib を用いているため Python が必要となっています。Python を使わない場合は gnuplot 等で代用してください。


#### Python ファイル（sir.py, seir.py)
SIR モデルおよび SEIR モデルのシミュレーションを行うプログラムとなっています。namespace 内のパラメーターは適宜変更してシミュレーションを行ってください。このテキストファイルに初期パラメーターを記述しておきます。  
##### 「SIR モデル（sir.py）」
|変数名|パラメータ|
|-----|--------|
|t|解析時間の最大（初期値は100）|
|dt|微小時間（初期値は 1 / t）|
|beta|感染症の感染率（初期値は0.001）|
|gamma|感染症の回復率（初期値は0.35)|
|S|シミュレーション開始時における感染症に対する非免疫保持者（初期値は950）|
|I|シミュレーション開始時における感染症の発症者（初期値は50）|
|R|シミュレーション開始時における感染症からの回復者（初期値は0）|

#### その他のファイルについて（「data-plot-cpp.py」,「sir, seir」,「sir.txt, seir.txt」）
##### 「data-plot.py」
解析結果のテキストファイルをプロットするための Python プログラムです。実行時にモデル名を小文字で指定してください。もし、Python を用いずにプロットする場合は特に必要ありません。

* ex) SIR モデル `python data-plot.py sir`

##### 「sir_c++.txt、seir_c++.txt, sir_python.txt, seir_python.txt」
SIR モデルおよび SEIR モデルの実行結果です。


# 解析結果の可視化について
今回、Python および C++ のそれぞれで同じモデルのシミュレーションを行いました。解析結果(「sir_c++.txt, seir_c++.txt, sir_python.txt, seir_python.txt」）の可視化は Python の可視化用の外部ライブラリ（Matplotlib） を利用しました。可視化に関しては、gnuplot や Excel など個人で自由に選んで貰えればいいです。
